# Interactive-Proteomics-Analytics-Web-App
Interactive web app for differential proteomics across EBV, EBV+AG490, CD40, and CNTRL at 6h/12h/24h: computes log2 fold-changes for pairwise comparisons, filters significant genes/proteins, visualizes clustered heatmaps, and intersects sets via an interactive Venn diagram to drive pathway analysis.
